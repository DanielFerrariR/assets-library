## Configuration

1. **Install these packages (prefer the listed versions):**

- node v20.10.0

2. **Install all dependencies**

```sh
npm install
```

3. **Start the development server**

```sh
npm start
```

4. **Commands**

```bash
# Install all dependencies
$ npm install

# Run production build
$ npm start

# Build for development
$ npm run dev

# Build for production
$ npm run build
```
